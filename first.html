HTML

HTML stands for HyperText Markup Language. It is the standard markup language used to create web pages. HTML elements are the building blocks of websites.

Table of Contents:-

History

  • Developed by Tim Berners-Lee in 1991
  • First version was HTML 1.0
  • Latest major version is HTML5

Structure

  • An HTML document usually contains:
  • A !DOCTYPE html tag declaration
  • html, head, and body tags
  • Elements like headings, paragraphs, links, and images
